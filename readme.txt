Модуль Яндекс.Турбо для товаров ocStore 2.3.0.2.3

+++++++++++++++++++++++++++++++++
Не использует модификаторов. Не заменяет системные файлы.
Дату публикации берет из дата модификации товара
Главное изображение размером 300х300
Описание как главное описание без html
Цену

+++++++++++++++++++++++++++++++++
Установка.
1. Загрузить модуль через "установка дополнений".
2. Войти в раздел Дополнения -> Каналы продвижения -> Яндекс.Турбо
3. Активировать модуль.

+++++++++++++++++++++++++++++++++
Использование.
Выбираете нужные разделы и включаете модуль. URL подсовываете в вебмастере яндексу.

Модификация v1.0
---------------------------------
Изменения:
+ Выбор категорий в выгрузке как для Яндекс.Маркет;
